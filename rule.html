[ 강월초등학교 6-5 웹사이트 규칙 ]



게시판 규칙에 대한 글입니다. 게시판의 규칙은 기본적으로 자유게시판 규칙에 준거합니다.

즉, 자유게시판에서 해서는 안되는 말은 게시판에서도 자제해주시는 것이 맞습니다.

자유게시판의 규칙의 주요 골자는 다음과 같습니다.

▶ 자유게시판 규칙

지나친 욕설 / 내상언급 / 소통의여지가 없는 일방적이고 무의미한 글 /

도배 / 지나친 친목 / 분란조성 / 수위오버 / 단속 / 카더라 / 타인 신상 언급 /

운영 및 이벤트에 대한 내용 / 이벤트 방문 관련 불협화음 등등에 대한 금지

타인의 기분을 상하게 하는 말(성희롱성 발언등) / 반말 사용 / 타 유흥사이트 언급 / Event or 후기관련해 공정성에 문제가 될 행동 / 지나친 혼잣말(특히 반말) / 채팅창 내 지나친 영업 / 기타 예의와 사회통념에 어긋나는 행동 등등에 대한 금지
